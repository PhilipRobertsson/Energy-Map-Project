# Energy Map  
**Mapping the Pulse of Power**

Exploratory tool for public public exhibitions with the primary goal to inspire personal environmental stewardship. This project also has a secondary target audience in the form of researchers, where it is meant to provide them with a creative model for experts to bridge the gap between complex data and public understanding.

This project has been based on the PhD project by Jinyi Wang titled EnergyViz, found here:

- [EnergyViz](https://github.com/Jinyi-Wang/EnergyViz/tree/main)

## How to run
Since no build for the tool is available yet, it can be run on localhost following these steps:

- Download the repository, and navigate to the folder, open the git terminal in it.
- Run ```npm install``` in the terminal to install dependencies, will create the ```node_modules``` folder.
- Once installed, run ```npm run dev``` in the terminal to start a local server.
- Navigate to the displayed url in preferred browser.
